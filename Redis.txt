基本配置：
    修改配置：sudo vi /etc/redis.conf
    守护进程：daemonize
    记录pid：pidfile
    端口：port
    绑定ip：bind
    启动：sudo service redis stars
    停止：sudo service redis stop
    查看进程:ps ajx|grep redis、
    连接：redis-cli

redis是key-value的数据，所以每个数据都是一个键值队
键的类型是字符串
值的类型为五种
    1.字符串string
    2.哈希hash
    3.列表list
    4.集合set
    5.有序集合zset
数据操作命令：可在redis中文官网
常用命令：
//string
    设置键值：set key value
    设置键值及过期时间，以秒为单位：SETEX key seconds value
    设置多个键值：MSET key value[key value ...]
    获取：get key    无值返回:nil  //和go一样
    获取多个值 mget key key ...

    运算：
    将key对应的value加1：INCR key
    将key对应的value加整数：INCRBY key increment
    将key对应的value减1：DECR key
    将key对应的value减整数：DECRBY key decrement

    追加：
    追加值：append key value
    获取值长度：strlen key

//键
    查找键，参数支持正则：keys pattern
    判断键是否存在：exists key [key ...]
    查看键对应的value的类型：type key
    删除键及对应的值：del key [key ...]
    设置有效时间：expire key time

//哈希hash
    设置单个属性：hset key field value  //field为属性名字
    设置多个属性：hmset key field value [field value]

    获取：
    获取一个属性的值：hget key field
    获取多个属性的值：hmget key field [field]
    获取所有的属性和值：hgetall key
    获取所有的属性：hkeys key
    返回包含属性的个数：hlen key
    获取所有的值：hvals key

    其他：
    判断属性是否存在：hexists key field
    删除属性及值：hdel key field [field ...]
    返回值的字符串长度:hstrlen key field

//列表list   //列表的元素类型为string
    设置：
    在头部插入数据：lpush key value [value ...]
    在尾部插入数据：rpush key value [value ...]
    在一个元素的前|后插入新元素：linsert key before|after pivot  value
              例：linsert py4 after 0 "hello"

    获取：
    移除并且返回key对应的list的第一个元素：lpop key
    移除并且返回存于key的list的最后一个元素：rpop key
    返回存储在key的列表里指定范围的元素：lrange key start stop

    其他：
    裁剪列表，改为原始集合的一个子集:ltrim key start stop
    返回存储在key里的list的长度：llen key
    返回列表里索引对应的长度：lindex key index

//set
    设置：
    添加元素：sadd key member [member ...]
    获取：
    返回key集合所有的元素：smembers key
    返回集合元素的个数：scard key
    其他：
    求多个集合的交集：sinter key [key ...]
    求某集合与其他集合的差集：sdiff key [key ...]
    求多个集合的合集：sunion key [key ...]
    判断元素是否在集合中：sismember key member

//zset
    设置：
    添加元素：zadd key score member [score member ...]
    获取：
    返回指定范围内的元素：zrange key start stop
    返回元素个数：zcard key
    返回有序集合key中，score值在min和max之间的成员：zcount key max
    返回有序集key中，成员member的score值：zscore key member

发布订阅：
    订阅频道：subscribe 频道名称
    取消订阅：unsubscribe 频道名称 //如果不写参数，表示取消所有的订阅。
    发布：publish 频道 消息

主从配置：
    设置主服务器配置：bind ip
    设置从服务器的配置：bind 从ip
                      slaveof 主ip port
    在master和slave上分别执行info命令，查看输出信息
    在master上写数据：set hello world
    在salve上读数据：get hello
